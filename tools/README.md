# Tools Directory

Utility scripts for data processing and export management.

## Scripts

### `generate_printable.py`
**Purpose**: Convert JSONL product data to printer-friendly HTML catalogs with dual format support and Bulgarian labels.

**Usage**:
```bash
# Traditional cards format
python3 tools/generate_printable.py --input exports/category/category.jsonl --output catalog.html --lang bg --format cards

# Compact table format for printer scaling
python3 tools/generate_printable.py --input exports/category/category.jsonl --output catalog.html --lang bg --format table
```

**Features**:
- **Dual Formats**: Cards (3-column detailed) or Table (10-column compact)
- **Print Optimization**: Cards use portrait A4, tables use landscape A4
- **Complete Data**: Includes product names, prices, quantities, codes, and images
- **Bulgarian Language**: Native labels and currency formatting  
- **Local Images**: Automatic path normalization for offline viewing
- **PDF Generation**: Optional PDF export (with Playwright)

### `reprocess_exports.py` 
**Purpose**: Re-fetch and update existing export data using enhanced extraction algorithms.

**Usage**:
```bash
python3 tools/reprocess_exports.py
```

**Features**:
- Updates all JSONL files in exports/ with improved extraction
- Regenerates combined printable HTML
- Preserves existing image files and metadata
- Applies rate limiting to respect site policies

### `generate_examples.py`
**Purpose**: Create sample HTML files from exported product data for testing/demonstration.

**Usage**:
```bash
python3 tools/generate_examples.py
```

**Features**:
- Generates example HTML files from select categories
- Demonstrates quantity extraction capabilities
- Creates visual samples for validation

## Integration

These tools are designed to work with:
- Export data generated by `ebag_runner.py`
- The enhanced scraper in `ebag_product_scraper/`
- Make targets: `make demo-single`, `make demo-full`
